# Day 2 - Linux Basics

## What is Linux?
Linux is an operating system used on servers, cybersecurity tools, cloud systems, and many IT environments.

## Why Linux is important in cybersecurity
- Many servers use Linux
- Cybersecurity tools often run on Linux
- Security analysts use Linux commands to investigate files, logs, and systems
- Cloud and DevOps jobs often require Linux knowledge

## Basic Linux commands

| Command | Meaning |
|---|---|
| `pwd` | Shows the current folder location |
| `ls` | Lists files and folders |
| `cd` | Changes directory/folder |
| `mkdir` | Creates a new folder |
| `touch` | Creates a new file |
| `cat` | Displays file contents |
| `clear` | Clears the terminal screen |
